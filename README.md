# Calculadora em C++

Este projeto é um **exercício de aprendizado e evolução** de uma calculadora simples, desenvolvida em **C++**. O foco é treinar técnicas de desenvolvimento e melhorar o código com o tempo, incluindo novas funcionalidades e melhores práticas de programação.

## Funcionalidades - Versão 1

- **Operações básicas**:
  - Soma
  - Subtração
  - Multiplicação
  - Divisão

- **Planejadas para versões futuras**:
  - Adicionar novas operações como exponenciação e raízes quadradas.
  - Implementação da calculadora com **Programação Orientada a Objetos (POO)**.
  - Múltiplas operações.
